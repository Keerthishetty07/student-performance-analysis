# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes student academic performance using data analysis and visualization techniques.
The goal is to identify trends, subject-wise strengths and weaknesses, and relationships between different subject scores.

---

## 🎯 Objectives

* Calculate subject-wise average scores
* Identify highest and lowest performing subjects
* Analyze correlation between subjects
* Visualize performance using graphs and heatmaps
* Extract insights about top-performing students

---

## 🗂 Dataset

The dataset contains student scores across multiple subjects.
It is used to perform statistical analysis and generate meaningful academic insights.

---

## 🛠 Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* VS Code

---

## 📈 Analysis Performed

* Average score bar graph
* Correlation heatmap between subjects
* Top 5 student performance analysis
* Identification of lowest and highest scoring subjects

---

## 🔍 Key Insights

* Some subjects show strong positive correlation, indicating linked learning patterns
* Average score comparison highlights weaker subject areas
* Top performers maintain balanced scores across subjects

---
